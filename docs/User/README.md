## 关于不同语言

* `Java`

  输入的代码实际上是`public static void main(String[] args)`中的内容，不能使用`import`.

* `Python`

  代码可以使用`import`.

* `C++`

  代码可以使用`#include`，但是必须包含主函数，即：

  ```cpp
  #include<string>
  using namespace std;
  int main(){
  	// Your codes here
  }
  ```
