# 数组(`array`)

## `Java`

```java
DataList data = new DataList(); // 新建一个DataList实例(必须)
// 支持的API
data.get(index); // 取值
data.swap(index1,index2); // 交换
data.size(); // 获取长度
```

## `Python`

```python
data = DataList() # 新建一个DataList实例(必须)
data.get(index) # 取值
data.swap(index1,index2) # 交换
data.size() # 获取长度
```
